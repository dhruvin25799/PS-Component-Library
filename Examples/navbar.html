NavBar (Submenu)

Navigation bar is key to a good UX of any website. Here we have made it responsive with submenus and made it mobile friendly.

  • Link
  • Submenu
    • Drop1
    • Drop2
    • Drop3

Usage

Use the class navbar on a nav element to use this component.

Use the class menu on a ul/ol to add links to the nav.

Use the class link-dropdown on an li to make it a hover dropdown style submenu. Inside that li now you can use ul/ol again.

On the first li you can use class navbar-brand so that during mobile view brand is displayed. Also add an hamburger icon with it to make it truly responsive.

Note: The menu is toggled with a display class when hamburger icon is clicked, this can be changed with your javascript.
