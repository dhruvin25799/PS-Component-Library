    Modal is used to alert information directly to the user instantly.

    This is some text inside Modal.

    Usage

    Use the class modal on a div to use this component.

    Use the class modal-body on a div inside the modal to put content in the body.

    Use the class modal-dismiss for the dismiss button.

    Note: The ID: myModal can be changed to target the modal according to your JavaScript.
