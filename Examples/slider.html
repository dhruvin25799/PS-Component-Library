Slider

Sliders are an important part of everything numbers related. A range slider helps to display and select ranges.

50

Usage

Use the class slider-container on a div to use this component.

Inside this div you can use input element with class slider to use this slider.

To get range value displayed, use a p with ID : rangeValue
