Grid

Grids are useful for many scenarios, where you want to have data displayed in tabular format. We also have a 2-item grid and a 3-item grid for most of your cases.

I am some text
I am red box

Usage

Use the class grid-container on a div to use this component.

Additionally use can use our extra classes two-item three-item to get a two-item and three-item grid.

Note : The grid has a 1rem gap.
