    Usage

    To use our cards, create a div with class card.

    For a horizontal card, add the class horizontal on the div the with card class

    Inside the div you can use classes card-header, card-image, card-content, card-cta
    to add those specific components inside a div.

    For text overlay on image, move the card-header class inside card-image class and add bottom-left to card-header div
